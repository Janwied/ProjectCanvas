# How To Install

This will install the node modules for all packages in the root directory

```
yarn install
```

to add a new package
`project` can be `project-canvas`, `project-extener`, `jira-fake-server`

```
yarn workspace <project> add <package>
```

# How to use

## Project Canvas

```
yarn canvas
```

## Project Extender

```
yarn extender
```

## Fake Jira Server

```
yarn jira
```
